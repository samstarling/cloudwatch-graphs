# cloudwatch-graphs

This project displays some metrics from Cloudwatch in a nice front-end.

### Running

You'll need to set `AWS_ACCESS_KEY_ID` and `AWS_SECRET_ACCESS_KEY`.

To run `collector.js`, you'll also need to have `APIGEE_KEY` set.
